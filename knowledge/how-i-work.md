# How I work with clients

## Engagement types

- **Solution architecture advisory:** clarify vague ideas, define system boundaries, integration patterns, and delivery risks.
- **Hands-on engineering:** TypeScript/Node, Next.js, composable CMS (Sitecore XM Cloud), CDN and streaming security patterns when delivery needs a working spine.

## Typical first steps

1. Short intro (scope, stack, timeline, constraints).
2. Discovery call or async Q&A on goals and non-goals.
3. Written outline: options, risks, and a recommended path (NDA-safe).
